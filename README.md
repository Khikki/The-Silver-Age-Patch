# Polymer Patch for The Silver Age

This is a patch for [The Silver Age](https://modrinth.com/mod/the-silver-age), making it work fully server side, with all of its functionality intact.

The Silver Age is required and not included within this mod.

This mod requires server resource pack, about which setup you can read here: https://polymer.pb4.eu/latest/user/resource-pack-hosting/

You can read more about [The Silver Age on it's Modrinth page!](https://modrinth.com/mod/rustic-delight)
